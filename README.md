# waw Pages part


## Template Creation
### Step 1
Create in 'server/pages/templates' new .html file
### Step 2
In the created file write the html-code for your template
### Step 3
Add name and file(.html) to part.json
example:
```javascript
"templates": [
...
	{
		"name": "My Template",
		"file": "mytemplate.html"
	},
...
]
```
### Step 4
Use template;)
